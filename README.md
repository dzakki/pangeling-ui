# pangeling-ui

A simple design system maintains brand consistency with predefined design guidelines.

## Installation

```sh
npm install pangeling-ui
```

## Usage

```tsx
import { HelloWorld } from 'pangeling-ui';

// ...

export default function App() {

  return (
    <View style={styles.container}>
      <HelloWorld />
    </View>
  );
}
```
